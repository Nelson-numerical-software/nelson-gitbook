



fix


fix

Round towards zero

## Syntax

- C = fix(A)

## Input argument

 - A - a variable

## Output argument

 - C - result of fix.

## Description


  <p><b>fix</b> returns an integer matrix made of nearest rounded integers toward zeros.</p>


## Example

```Nelson
fix(pi)
```
