



now


now

Returns current date under the form of a Unix hour.

## Syntax

- n = now()

## Output argument

 - n - a double.

## Description


  <p><b>now()</b> returns the current date and time as a serial date number.</p>


## Example

```Nelson
datevec(now())
```
