



double


double

Converts a variable to double precision type.

## Syntax

- D = double(V)

## Input argument

 - V - a variable.

## Output argument

 - D - a double.

## Description


  <p><b>double(V)</b> converts to the double-precision type.</p>


## Examples

```Nelson
double('Nelson')
```
```Nelson
A = single(pi)
B = double(A)
B - A
```
