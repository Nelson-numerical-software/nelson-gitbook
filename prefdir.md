prefdir

Return the preferences directory used by Nelson.

## Syntax

- pref_path = prefdir

## Output argument

 - pref_path - a string: the preferences directory

## Description


  <p><b>pref_path = prefdir()</b> returns the preferences directory used by Nelson.</p>


## Example

an example
```Nelson
cd(prefdir)
```
