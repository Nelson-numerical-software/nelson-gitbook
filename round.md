round

Round to nearest integer

## Syntax

- C = round(A)

## Input argument

 - A - a variable

## Output argument

 - C - result of round.

## Description


  <p><b>round</b> rounds the elements to the nearest integers.</p>


## Example

```Nelson
round(pi)
```
