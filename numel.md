



numel


numel

Number of elements.

## Syntax

- nbel = numel(M)

## Input argument

 - M - a variable

## Output argument

 - nbel - the number of elements.

## Description


  <p>Return the number of elements in the object M.</p>


## Example

```Nelson
numel(ones(3, 0))
numel(ones(3,4))
numel(ones(3,4,5))
```
