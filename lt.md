



lt


lt

less than, < operator.

## Syntax

- C = lt(A, B)

## Input argument

 - A - a variable
 - B - a variable

## Output argument

 - C - result of lt(A, B)

## Description


  <p><b>C = lt(A, B)</b> returns a logical array with elements set to logical <b>true</b> A is less than B.</p>
  <p/>


## Examples

```Nelson
eye(2,2) &#60; ones(2, 2)
```
```Nelson
0 &#60; i
```
```Nelson
'Nelson' &#60; 'Noslen'
```
```Nelson
'Nelson' &#60; 'l'
```
```Nelson
lt(0.8 - 0.6 - 0.2, 0)
```
