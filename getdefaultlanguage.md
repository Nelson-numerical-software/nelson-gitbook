getdefaultlanguage

Returns the default language used in Nelson.

## Syntax

- lang = getdefaultlanguage()

## Output argument

 - lang - a string: 'en_US' by default.

## Description


  <p><b>getdefaultlanguage</b> returns the default language used by Nelson.</p>


## Example

```Nelson
getdefaultlanguage()
```
