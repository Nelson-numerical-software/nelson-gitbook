

# actxcontrollist

Get available ActiveX controls.

## Syntax

- l = actxcontrollist()

## Output argument

 - l - a 1 by 3cell of strings.

## Description


  <p><b>l = actxcontrollist()</b> returns list of available controls</p>


## Example

```matlab
l = actxcontrollist()
```
