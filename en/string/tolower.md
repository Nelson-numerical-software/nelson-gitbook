

# tolower

Lower case conversion.

## Syntax

- res = tolower(str)

## Input argument

 - str - a string.

## Output argument

 - res - a string lower case

## Description

<b>tolower</b> converts a string to lower case.

## Example

```matlab
tolower('NelSon')
```
