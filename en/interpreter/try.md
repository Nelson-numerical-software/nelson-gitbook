# catch

try/catch statement.

## Syntax

- try, statements_1, catch, statements_2, end

## Description


  <p><b>try</b> and <b>catch</b> statements are used for error handling and control in files.</p>


## Example

try/catch in a script file
```matlab
try 
error('an error')
catch
  disp('error catched')
end
```
