nelsonroot

Returns Nelson's root folder.

## Syntax

- nelson_path = nelsonroot

## Output argument

 - nelson_path - a string: the root folder of Nelson.

## Description


  <p><b>nelsonroot</b> returns the root folder of Nelson.</p>


## Example

```Nelson
pwd
cd(nelsonroot)
pwd
```
