



userdir


userdir

Returns the current user's path.

## Syntax

- userdir()
- p = userdir()

## Output argument

 - p - a string: current user directory.

## Description


  <p>Returns the name of the user's directory.</p>


## Example

```Nelson
r = userdir()
```
