pathsep

Return the search path separator character for the current platform.

## Syntax

- res = pathsep()

## Output argument

 - res - a string: ';' or ':'

## Description

<b>pathsep</b> returns ';' on Windows and ':' on others platforms.

## Example

```Nelson
A = pathsep
```
