



ischar


ischar

Return true if variable var is a char array.

## Syntax

- res = ischar(var)

## Input argument

 - var - a variable

## Output argument

 - res - a logical: true or false

## Description

<b>ischar</b> returns a logical 1 if the argument is a char array and a logical 0 otherwise.

## Examples

```Nelson
A = 3;
res = ischar(A)
```
```Nelson
B = 'NelSon';
res = ischar(B)
```
```Nelson
C = [1 ; 3];
res = ischar(C)
```
