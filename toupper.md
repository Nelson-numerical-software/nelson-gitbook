toupper

Upper case conversion.

## Syntax

- res = toupper(str)

## Input argument

 - str - a string.

## Output argument

 - res - a string upper case

## Description

<b>toupper</b> converts a string to upper case.

## Example

```Nelson
toupper('NelSon')
```
