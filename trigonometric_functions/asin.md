asin

Computes the inverse sine in radians for each element of x.

## Syntax

- res = asin(x)

## Input argument

 - x - a numeric value

## Output argument

 - res - a numeric value

## Description

<b>asin</b> computes the inverse sine in radians for each element of x.

## Example

```Nelson
A = eye(3, 3);
res = asin(A)
```
